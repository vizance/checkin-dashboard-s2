## 🚨 常見地雷（從第一屆經驗）

1. **姓名不一致**：學員名單是「張三」，Form 是「張 三」（多空格）→ 系統會抓不到。輸入名單時要乾淨
2. **忘記關測試模式**：前後端都有 `TEST_TODAY_DATE`，兩個都要是 `null` 才是正式模式
3. **試算表共用權限**：必須是「知道連結的任何人 - 檢視者」，否則儀錶板會顯示「載入失敗」
4. **第一次推 GitHub Pages 要等**：通常 1-2 分鐘才會生效，不要急
